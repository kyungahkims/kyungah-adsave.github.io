    • 2. 사이트 내 여러 개 광고지면을 사용할 때 지면 고유번호 또는 지면코드 정보를 중복되지 않게 설치해야 합니다.
    • 3. APP의 경우 코드설치 후 즉시 확인 불가능하며, 익일부터 설치결과 확인 가능합니다.
    • 4. 코드 설치과정에서 문의사항이 있으신 경우 고객센터로 문의 주시기 바랍니다.
    • 5. 코드설치 후 지면URL 및 지면 상세설명을 입력해주시면 빠르게 심사 가능합니다.

    다음 광고 코드를 사용하여 지면을 활성화하고 광고 게재를 시작하세요.

    지면이 활성화되면 광고가 표시됩니다.
    1. 1. 위의 코드를 복사하세요.
    2. 2. 사이트 HTML에서 광고 노출을 원하시는 영역 내 코드를 붙여넣으세요.
    3. 3. 코드 설치를 완료하신 후 게재상태를 변경해주세요.
