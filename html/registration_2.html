사이트 정보

  • 사이트 형태
    • 사이트가 노출되는 기기 형태에 따른 설정
    • PC = 웹 / MOBILE = 모바일 / 반응형 = 웹
  • 반응형 웹 지원 여부
    • 하나의 웹 사이트로 다양한 기기에 최적화된 화면을 노출하는지에 대한 설정
  • 사이트 통신 규약
    • 사이트가 SSL 인증서 사용 여부에 따른 설정
  • 사이트 URL
    • ex) www.enliple.com

지면 영역

  • 지면 구현 방식
    • 일반 : PC or MOBILE 기본 값
    • SDK(Software Development Kit) : MOBILE APP 지면 세팅용
    • RTB(Real Time Bidding) : 실시간 경매 지면
  • 지면 형태
    • 지면의 노출 형태
  • 지면 사이즈
    • 사이트에 광고가 노출될 영역의 사이즈를 선택합니다.

지면 정보

  • 지면명
    • ex) 매체명_웹_메인_하단_320x50
  • 지면 상세 설명
    • 네이티브 기사 연동시 매체사명 노출

지면 설정

  • 지면 카테고리
    • 지면 카테고리는 광고 이벤트에서 지면 매칭시 성향에 맞는 지면 선택에 활용 됩니다.
  • 오디언스 카테고리
    • 지면 카테고리 세분화 카테고리

정산 정보

  • 매체사 정산 방식
    • 매체가 지면 수익금에 대해 정산 받을 정산 방법 설정
      RS(Revenue Share) : 매체피 수익 분배 퍼센트 값 입력
    • GT(Guarantee) : 보장된 금액 (원)으로 값 입력
      CPM(Cost Per Mille) : 1,000번 노출 당 비용 (원)으로 값 입력
    • CPC(Cost Per Click) : 1회 클릭 당 비용 (원)으로 값 입력
      서비스 : 무료 지면
      비딩 : RTB 정산 방식이며 단가 0으로 설정
